VSTS-Confluence Integration

In order to track completed tasks in VSTS easily, this program will create a blog post in Confluence with the details of that task. It uses the REST APIs of both Microsoft VSTS and Atlassian Confluence as well as the power of C# and .NET framework.

Using the client-side app, you can join a server and chat with those on it. It uses TCP/IP sockets, the tkinter GUI library and a kickstart from here.
